> I can only access the internet using a proxy server with a specified port. My question: what should I change in my .fetchmailrc settings to be able to fetch mail using fetchmail?

Globe Trotter,

you have not specified the kind of proxy that you must use.

Unless your proxy is a SOCKS-type proxy (the only proxy that fetchmail
accesses natively - details in the manual page), you will need to use a
"plugin", because fetchmail itself does not currently use HTTP proxies
for "CONNECT", such as squid or a proxy that is usually used for web
access. It would need to support https, though.

"plugin" is also the name of fetchmail's option, and is being used to
connect to POP3/IMAP servers; "plugout" is the complement for connecting
to SMTP servers.

You would have to find a program that connects to your proxy, configures
it to relay your traffic properly, and talks to fetchmail through the
stdin and stdout file descriptors.  If you can find such a proxy helper
program, configure fetchmail with plugin and - if needed - plugout, and
you should be set.

Without further indication of what your proxy server exactly is and what
protocol it uses, we cannot help you further than this.  Feel free to
ask again with more detailed information.

Hope that helps.

Matthias
