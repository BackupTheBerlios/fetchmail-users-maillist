And what were those errors?  If you require assistance from list members
then you need to provide sufficient information for us to help you.  If
you are just spreading FUD then go away.

I have used --silent in the past and it worked fine.  These days I have

  set no syslog
  set logfile <path-to-log-file>

in my .fetchmailrc file.  NB: the file must exist before fetchmail is
started for it to write to it, as noted in the manual page.

And, BTW there is no version 2.6.3.19.  The latest version is 6.3.19.
