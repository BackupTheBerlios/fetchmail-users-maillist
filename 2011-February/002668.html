Is there a particular reason why you're setting forcedcr?
It should not normally be needed for use with procmail.

Note that you can run fetchmail on individual servers, like:

fetchmail --nodetach --daemon 0 -vv pop.live.com

(the last part is one or more server names from poll or skip lines)
