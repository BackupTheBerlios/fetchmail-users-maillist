no offense intended, but can I ask you to _read_ the given documentation, particularly these items:

- screen or log file output

- manual page

- FAQ, particularly section G3.

Reason I'm asking is that there is sufficient documentation in place to help you solve half of your problems yourself.

Also note that none of the other list readers is capable of clairvoyance, meaning that without seeing your .netrc file nobody can do anything to help. Remember to mask the passwords when posting configuration files.

I checked the certificate, and it verifies the pop3.live.com certificate fine. I suppose you either haven't run "c_rehash /home/jude/.certs", or the version of c_rehash doesn't match the library that fetchmail has been linked against.
