Jude DaShiell schrieb am 2011-02-12:

> On slackware systems /usr/share/curl/ has ca-bundle.crt.  I found several 
> Thawte certificates in that file and an Equifax certificate too.  Would 
> use of anything in that file be at all helpful?

Hi Jude,

Quite likely so, providing that the Thawte(?) certificate that signed
the gmail server certificate is included.  To use it, a *recent* fetchmail
version (the feature was a recent addition) can be configured to look in
that place by --sslcertfile /usr/share/curl/ca-bundle.crt (omit the --
if putting this in the rcfile).  Note that outdated fetchmail versions
don't offer that option.

Note also that using such bundled certificate files
(a) does not require to use c_rehash, as opposed to directories that use
one-file-and-one-symlink-per-certificate (with --sslcertpath)
(b) loads all certificates on startup.  Not a problem on halfway modern
computers, but possibly slow on ancient hardware.

HTH.
