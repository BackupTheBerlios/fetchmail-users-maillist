Good E-Mail programs under Windows (YES), like Eudora, have a size-limit
function too.  If a Mail is bigger as the given value, the E-Mail-Program
download ONLY the Header and Add a text to it which telle the usr that
the Message had exceeded the fetchsize limit and show the real size.

So in fetchmail they can add something like:

----8<-----------------------------------------------------------------
From: Michelle Konzack <linux4michelle at freenet.de>
To: Nico Golde <nico at ngolde.de>, 416282 at bugs.debian.org
Subject: Re: Bug#416282: Fetchmail oversized-messages warning could say more

!!! FETCHMAIL SIZE LIMIT WARNING !!!

The original messages exceeded the configured fetch size of
10 kBytes and was not downloaded.  If you want to download it,
please use fetchmail manualy with the option "--limit 0".
