missing something.

Debian/squeeze supports running a systemwide fetchmailrc file by
running fetchmail in daemon mode as user "fetchmail."  I have a home
server with a number of different users .fetchmailrc files so I thought
it would be a nice win for me to put everything into one fetchmailrc
file.  This is made more attractive because there are multiple accounts
