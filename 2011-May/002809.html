I think 6.3.9-rc2 is packaged by Debian, hence inherited by Ubuntu, and
it's still the "current" version for Ubuntu lucid, which is their
long-term support version 10.04.  Ubuntu hardy (8.04 LTS) and dapper
(6.06 LTS) have even older versions, 6.3.8 and 6.3.2.

The latest Debian stable release is somewhat up to date, using 6.3.18.
It has some quirks fixed in 6.3.19, but close enough.

If there is someone who maintains a ppa with up-to-date fetchmail
releases, please speak up.
