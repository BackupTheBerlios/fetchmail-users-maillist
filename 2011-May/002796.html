Hi Yunfan,

Be sure to deinstall the existing distribution version (particularly on
dpkg-based distributions), then install and run the newly built
fetchmail executable - likely you need /usr/local/bin/fetchmail, no
longer /usr/bin/fetchmail or just "fetchmail". You can confirm the
version with an additional (upper case) "-V".  It works for me, although
on a different IMAP server (not Google mail):
