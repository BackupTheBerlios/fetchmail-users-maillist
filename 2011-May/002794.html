Hi Jun-Fang,

we've recently changed fetchmail to speed the IMAP code up - however,
that is code that will probably not be released for Ubuntu 10.04.

If you're happy with building from source and bootstrapping the code,
grab the git master branch from
<http://gitorious.org/fetchmail/fetchmail> and run

autoreconf -isvf
./configure --with-ssl
make -j4
make install

and then use /usr/local/bin/fetchmail. I am oblivious to the required
packages though.

You may want auto-apt to assist with installing them (though I've never
used it myself).
