Well, it took some time, but now fetchmail-6.3.19 is available in
the Debian lenny-backports-sloppy distribution.  Any Debian Lenny user
may install it by following the instructions on the backports page:
http://backports.debian.org/Instructions/ - use "lenny-backports-sloppy"
instead of "squeeze-backports" for the distribution, set the pinning
priority to 200 as described there, and everything should work just fine.
