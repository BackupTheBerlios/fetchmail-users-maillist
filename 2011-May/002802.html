The patch is attached to the mail again now. To apply the patch, change your directory to the fetchmail checkout directory, save it there, and run this command:

git am 0001-Do-not-print-skipping-message-for-old-messages-even-.patch

[You will have to give your saved file name in the above command]

After that, build, install, and run fetchmail again.
