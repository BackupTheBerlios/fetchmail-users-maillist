Am 18.02.2009, 13:37 Uhr, schrieb shacky <shacky83 at gmail.com>:

> I'm using fetchmail to fetch some accounts' mails and to fetch all  
> messages.
> But I need to keep the e-mail messages, only for some accounts, and to
> delete them after some days (one week, for example).
> How I can configure fetchmail to behave like this?

I'm sorry, that's not currently possible in fetchmail. The contrib/  
directory may contain scripts that perform the "delete messages" part, or  
you may look into alternative packages (getmail for instance has such a  
feature) - but beware if you need to rely on SSL security: not all  
packages (namely, getmail again) can support strict certificate  
verification (= detect man-in-the-middle eavesdropping).

http://www.fetchmail.info/fetchmail-FAQ.html#G5
