I'm sure this question has a simple answer. I've been using fetchmail
for years and never had the need to poll more than one mail server. Now
I'd like to poll two different pop accounts. One I poll every 2 minutes
and the other I'd like to poll every 90 minutes. I have this in
my .fetchmailrc:

set postmaster cpollock
set no bouncemail
set no spambounce
set daemon 120
set syslog

and then the account settings for the account I'm using now.

Only one daemon process is permitted per user; in daemon mode, fetchmail
sets up a per-user lockfile to guarantee this. 

Reading the above from the manpage I assume that 'user' is me logged
into this machine? The commandline I use to run fetchmail is:

fetchmail -v --nosyslog --logfile /home/chris/fetchmaillog --uidl -m
procmail.
