Xin Li:

> should i need to recompile fetchmail with some ipv6 patch?

There's no need to recompile fetchmail for IPv6 if:

1. it was compiled in an IPv6 capable compilation environment

2. the operating system you will run fetchmail on is IPv6 capable (including resolver).
