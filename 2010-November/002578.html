The SMTP Authentication is a tough nut to crack. I had a similar
problem with a Postfix server. I had to go through all sorts of hurdles
to get my Postfix server configured to accept mail from fetchmail
while still keeping the rest of the system secure. I had problems with
both TLS {STARTTLS} and SMTP authentication.

You are going to have to get the Exchange server configured to accept
mail from fetchmail sans authentication. Perhaps at some point in the
future the ability the authenticate will be configurable in fetchmail.

You might also want to find out the size limit on the Exchange server
and have fetchmail match it.

Good luck!
