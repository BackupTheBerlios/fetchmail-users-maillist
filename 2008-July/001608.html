Well - it may suffice to install the fetchmailconf.py from fetchmail's
source directory, and optionally compile it to bytecode (this isn't
required) - given that fetchmailconf isn't usually run frequently, you'll
probably get away without even compiling it to .pyc or .pyo.

If you want to compile the script, look at py-compile (a script in
fetchmail's source directory); a reasonably complete Python install should
contain py_compile.* (the Python standard library module).
