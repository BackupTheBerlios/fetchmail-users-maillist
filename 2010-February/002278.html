> Greetings,

> Since a short while I started using fetchmail, mutt and postfix on my  
> Mac. It get's all the email from a POP3 server but deletes the messages  
> on the server, regardless the 'keep' option (I think) I'm using...

> fetchmail -a --uidl -v -f $HOME/.fetchmailrc -K $EXTRAARG >/dev/null 2>&1

Please see the fetchmail manual for -K. Note that options are *case  
sensitive*.

> If I check the configuration with fetchmail -V it says '(--keep on)',  
> but still :

Try "fetchmail -a --uidl -v -f $HOME/.fetchmailrc -K -V" and add whatever  
would be in $EXTRAARG.

Also note your fetchmail version is outdated. 6.3.14 is current.
