I use fetchmail to access numerous accounts, including some 'gmail' ones also.
My configuration for accessing 'gmail' is slightly different though.

# Configuration created Sun Jul 30 19:38:15 2006 by fetchmailconf 1.52 $Revision: 4740 $
set logfile "/usr/home/ges/logs/fetchmail.log"
set postmaster "postmaster"
set bouncemail
set spambounce
set properties ""
set daemon 900
poll pop.gmail.com with proto POP3 and options uidl
    localdomains seibercom.net

 user 'gerard.seibert at gmail.com' there with password 'secret' is
'gerard-gmail' here options fetchall forcecr pass8bits ssl mimedecode


The only real difference is that I do not give it a specific port number.
