I've been using fetchmail through a router for as long as I've been
using fetchmail, without problems.

The range of possible causes is not small, but can be reduced to a problem with:

1) The ISP
2) The router
3) Your local network configuration

You can somewhat rule out (3) if the host that runs fetchmail and
fetchnews also runs the other programs you refer to.

You can rule out (2) if you can swap the router for a different brand
that others know to work.

You can rule out (1) by switching ISP ;)
