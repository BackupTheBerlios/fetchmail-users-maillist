Is this router some kind of modem? You might be hitting the issue
briefly mentioned in FAQ X7. If your router has an escape character of
"+" and a mail contains "+ + +" (*), the router may intercept the text
as a modem escape sequence. Your options in that case are:

a) Disable the modem escape character, or
b) Change the modem escape character to some non-ascii character which
does not occur in mails, or
c) Download using ssl! (**)

-- 
Sunil Shetye.

(*) Without those spaces. If those spaces are not added, you may not
get this mail too!

(**) Only if your mailserver supports it. No modem reconfiguration
required.
