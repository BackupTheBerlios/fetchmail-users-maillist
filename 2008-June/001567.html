Michelle Konzack wrote:

> I run my regular fetchmail for my "standard" mailboxes  all  15  minutes
> but since I have installed a seperated tool which require fetchmail  and
> must run all 5 minutes I am running into trouble.

No you aren't. You can use one configuration and set "interval 3" for the server that requires the 15 min interval.

> Since all two  fetchmail  instances  run  from  a  script,  how  must  I
> manipulate it, that I can run more then one instance of fetchmail?

You're not supposed to, see above.

If you must, create a subdirectory, say $HOME/fetchmail-someserver, create a file fetchmailrc there (sans leading dot) and use the FETCHMAILHOME environment variable to point to fetchmail-someserver:

env FETCHMAILHOME=$HOME/fetchmail-someserver fetchmail # add other options as needed
