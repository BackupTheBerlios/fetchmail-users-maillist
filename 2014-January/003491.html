>> Download the tarball.

>> 
>> ./configure --prefix=/opt/fetchmail --with-ssl
>> [add other options as you see fit,
>> --help=short will list special options,
>> --help will list all options]
>> 
>> make check
>> make install   (this is the only operation that requires privileges)
>> 
>> Try: /opt/fetchmail/bin/fetchmail
