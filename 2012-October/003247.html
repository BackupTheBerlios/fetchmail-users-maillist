Very simply you need to configure an SMTP server on S and then
configure your mail client on C to use it. For the first, follow the
instructions for the SMTP server you've chosen and if you need help
use the help facilities for it. For the second, check the
documentation for your client and if you need help use the help
facilities for it.

(http://www.postfix.org/lists.html), so I can't help you with Postfix
and you've said nothing about your choice of client so I can't point
you to where to get help. Google should however be able to help you
with that ;)
