Absolutely, it has no capability to write any format of mail by
itself. All it can do is hand the problem over to something else -
since the format needs to be in one that your local IMAP/POP3/whatever
server supports and rather than being able to write every possible
format correctly (and somehow magically work out what that is), it is
much, *much*, easier to leave it to your local mail server to sort
out.
