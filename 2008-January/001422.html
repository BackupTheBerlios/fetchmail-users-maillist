> I'm using fetchmail with a multidrop POP3 mailbox and it works fine. But now and then somebody is complaining about missing emails. I've read the manual pages over and over but couldn't find any option to add this information to the logs. Is this feature available or not?

Not from fetchmail.  You should however find this information in your
SMTP server's logs.  If you need more you'll need to run fetchmail in
verbose mode I'm afraid (and be prepared for the large volume of logs
that results).
