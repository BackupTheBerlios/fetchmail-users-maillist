Greetings,

if fetchmail is started on Debian, the default init script appears to
support a "debug-run", which starts fetchmail in foreground mode,
logging to console, with verbose info -- this can help debug fetchmail
in some cases.

I hope we all remember this option next time a Debian user has trouble.
