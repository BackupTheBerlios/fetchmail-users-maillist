I am using pop3. Here is the content for fetchmailrc:

poll pop.gmail.com with proto POP3 and options no dns
user '__' there with password '__' is '__' here  options ssl
sslcertck sslcertpath /home/__/.certs/


Command line is simple:       fetchmail -vk
