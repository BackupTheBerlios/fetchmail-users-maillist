Hi Martin,

Not currently, sorry.  Fetchmail does not decode/present that
information, or have options to limit/select the ciphers.  I have added
that to the 7.X TODO.txt list, available from the Git repos.

Jason and Martin,

If 6.3.21 is the most current version for Cygwin, please see to
building/uploading a new version for Cygwin, or failing that, Martin,
you should build a current fetchmail version yourself; 6.3.21 lacks
several security and critical bug fixes (memory leaks plugged, data loss
in mimedecode).

<http://gitorious.org/fetchmail/fetchmail/blobs/raw/6b363d166c9eb8354176b796f2fed68d538b7a4a/NEWS>
