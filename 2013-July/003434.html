> I need a quick fix for this problem. I loose too many time with this shit.

> Let’s try your no dns option, but I don’t see any man page about what

> option ??

In the fetchmail manual page (you do have that, right?):

       Here are the legal server options:

       Keyword          Opt   Mode   Function
       ─────────────────────────────────────────────────────────────────
       ...
       dns                    m      Enable  DNS  lookup  for multidrop
                                     (default)
       no dns                 m      Disable DNS lookup for multidrop

The "m" means that this is an option that will only have effect in
multidrop mode, just like its description says.  The fact that it is in
the "server options" section of the manual page means that you put it
in, well, your server definition.  The "no dns" text in the "Keyword"
section means that, well, you add "no dns" to your server definition
(the "poll" line).
