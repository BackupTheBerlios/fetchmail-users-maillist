Errr, it shouldn't really have been downloaded "off the server on the
Internet"; can you post several more lines of the fetchmail log,
immediately following these?  If there is no "DELE" command sent to
the POP3 server in the following lines, then the mail should still
be there, waiting for you to fetch it.

Or am I assuming too much?  Is this not a POP3 server?  In any case,
a couple more lines from the fetchmail log should help - you cut it off
at almost the worst place possible, to keep us all in suspense :)
