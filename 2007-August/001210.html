fixed in 6.3.  Effectively it's a new feature, and one that's not
necessarily trivial.

The suggestion was that any invalid emails would be forwarded to the
postmaster account, as a plain text attachment.

The reason for the current behaviour is that there's no point in
handing the email off as-is to any mail server - any sensible mail
server would simply reject them.
