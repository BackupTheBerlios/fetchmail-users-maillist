Please upgrade to 6.3.19

> set daemon 120
> set postmaster "postmaster"
> set bouncemail
> set logfile /var/log/fetchmaillog
> set no spambounce
> set properties ""
>
>
> poll pop.mydomain.com
>        proto pop3
>        via pop.mydomain.com
>        timeout 60
>        envelope Delivered-To
>         user "multipopuser at mydomain.com"
>        pass "xxxxx"
>        is *
>        fetchall

Looking at the headers of your sample email it is clear that the
envelope header is not Delivered-to but Envelope-to. Try:

envelope Envelope-to
