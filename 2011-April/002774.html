Please upgrade - not only are you running a pre-release version of
fetchmail, but you're many versions behind current. While it may not
resolve your current problem, it will resolve other problems and fix
many security holes.

> Now i am using envelope "Envelope-to". All email from TO dan CC are okay.
>
> The problem is that fetchmail cannot fetch header from BCC and mailing list email.
>
> And then I analyzed the header, and i found that all email from BCC and mailing list email flagged with "X-Apparently-To".
>
> How can fetchmail read two header, so if header envelope-to are not found then it's automatically found header X-Apparently-To?

I'd suggest you talk to your mail provider and ask them to fix their
broken setup. Their mail server should only be using a single header,
not 2 different ones depending on how the email was sent. Fetchmail
does not support multiple Envelope headers.
