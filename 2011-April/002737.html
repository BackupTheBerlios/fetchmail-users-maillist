Hi there,

first, please see Steve's reply (thanks, Steve).

I don't currently have resources or interest to even try supporting this
over POP3.  Server implementations and behaviour are beyond my control,
and fetchmail is doing more or less all it can, and there is no
guarantee we will succeed if we try harder or differently.

Such efforts are just way down the list of priorities, and chances are
that I never get around to it.

I will - later - make sure we can fetch over IMAP without marking seen,
because there is protocol support for that in IMAP (but not in POP3).

For details, if someone cares to read technical stuff, see

http://tools.ietf.org/html/rfc1939 - for POP3
http://tools.ietf.org/html/rfc3501 - for IMAP.

HTH
Matthias
