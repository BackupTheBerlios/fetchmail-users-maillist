Greetings,

I have just released fetchmail 6.2.9-rc4, yet another release candidate
for 6.3.0, after minor fixes to -rc3 became necessary.

Changes since -rc3 are:
* Fix MacOS X compilation failures in sink.c (ru_*time has incomplete type).
  Berlios Bug #4725. Matthias Andree.
* Fix "auth ntlm" to send AUTH NTLM (rather than AUTH MSN). Add "auth msn"
  officially. Reported by Yves Boisjoly. Matthias Andree
* Expunge between IMAP folders when polling multiple folders.
  Sunil Shetye. (MA)
* Fix IMAP expunged message counting. Sunil Shetye. (MA)

I seek everyone to test it out, report remaining bugs, update outdated
translations or send patches for documentation or report inconsistencies
(including formatting!) in the documentation.

The software is available from:
<https://developer.berlios.de/project/showfiles.php?group_id=1824&release_id=6617>
<http://home.pages.de/~mandree/fetchmail/>
