These are the settings I use to access a 'hotmail' account.

poll pop3.live.com with proto POP3 service 995 timeout 30 and options
uidl user 'USER at hotmail.com' there with password 'PASSWORD' is 'USER'
here options forcecr dropdelivered  smtpaddress 'hotmail_.com' ssl
sslcertpath '/usr/local/etc/postfix/certs/' sslfingerprint
'26:3A:E1:E1:29:94:CB:D8:97:08:4D:1D:22:07:8F:AD'

I am not sure why you would be using port 587 though. That is for SMTP
not POP3. Hotmail does not work properly with TLS on port 110. At least
it didn't in the past. You have to use port 995 unless something has
changed. As you might notice, I am using Postfix with Fetchmail, hence
the requirement for the "dropdelivered' flag. You probably would not
need the 'smtpaddress' flag either.

As always, YMMV
