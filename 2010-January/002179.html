> Hi Aljoscha,
>
> Note that you may need to upgrade to 6.3.13 if Gentoo backported the 
> security fix from 6.3.12 to 6.3.11-r1; and you will have to configure 
> the softbounce option properly. See the manual page for details.
>
> If that does not help, please provide a full report as shown in 
> <http://www.fetchmail.info/fetchmail-FAQ.html#G3>, notably the log is 
> missing from your report.
>
>
> Please also note that addresses with "-" as the first character are 
> valid (see RFC 5321, RFC 5322), it's only a Postfix default setting 
> that causes these addresses to be rejected (to protect the innocent on 
> the assumption that many scripts will be foolish enough not to escape 
> and quote things properly and miss the "--" argument separator).
>
> HTH
