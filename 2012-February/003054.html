> I have a file which appeared when I had a fault in my postfix
> setup & it looks like an mbox file of a number of emails. I have
> tried to convert it to Maildir using mb2md but this reports it is
> not an mbox format. I would like to add these to mydovecot system,
> ideally by re-feeding them into postfix but any way would do.

Some time back I had a similar problem.  KMail was my mail client, and
it could read either format, so I created a folder within KMail with
the mbox format to use as an emergency folder.  It's long time ago, so
I can't remember with any accuracy, but IIRC I simply copied the mbox
file into the folder/directory created in this way, after which I
could read it in a mail client.  From there it was easy to pull the
mail into the wished-for maildir folders.

Hope that helps
