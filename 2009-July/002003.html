The reason I asked for your OS is because some have /etc/nsswitch.conf
that defines how name resolution works.  When you use a standard name
resolution it uses the methods defined there (eg, if you type "ping"
or "telnet").  If you use something like nslookup, dig or host then it
explicitly uses the entries in /etc/resolv.conf for DNS resolution.
