Try syslog-ng. It allows you to create separate log files for just
about anything, including the program that creates the original
message.  You can even have it split log files based upon the date,
time or anything else.

For instance, the following template will allow you to split logfiles
by host, date, time program and facility:

destination d_logfile {
file("/var/log/$YEAR/$MONTH/$DAY.$HOST.$PROGRAM.$FACILITY.log"
template("$FULLDATE $HOST $PROGRAM[]: ($LEVEL) $MESSAGE\n")); };

One log file per day, per host, per program and per facility ;)
