here's my config:

# General
set postmaster "postmaster at ourdomain.de"
set bouncemail
set no spambounce
set properties ""
set syslog
set daemon 180

# Accounts
poll mailserver.hoster.de with proto POP3 interval 3
         user "user at ourdomain.de" there with password "s3cr3t" is 
"user at ourdomain.de" here ssl
