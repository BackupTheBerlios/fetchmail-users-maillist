Hi,

i have this fetchmailrc:

# General
set postmaster "postmaster at mydomain.de"
set bouncemail
set no spambounce
set properties ""
set syslog
set daemon 300

# Accounts
poll mailserver.myprovider.de with proto POP3 interval 3
         user "user1 at mydomain.de" there with password "passwd1" is 
"user1 at mydomain.de" here ssl
poll mailserver.myprovider.de with proto POP3 interval 3
         user "user2 at mydomain.de" there with password "passwd2" is 
"user2 at mydomain.de" here ssl
poll mailserver.myprovider.de with proto POP3 interval 3
         user "user3 at mydomain.de" there with password "passwd3" is 
"user3 at mydomain.de" here ssl

works fine.
