Under the description of the sslfingerprint option, an example is provided
showing how to use openssl to extract the fingerprint.

I found that the -md5 option had to be included on the openssl command line to
obtain an MD5 fingerprint; otherwise, openssl defaulted to SHA-1. Thus the
example should be
openssl x509 -in cert.pem -noout -md5 -fingerprint
