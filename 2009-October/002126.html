Greetings,

I am announcing the release of fetchmail 6.3.13.  This new stable
version of fetchmail fixes a bug introduced into 6.3.12 that would treat
all SMTP errors as temporary, regardless of the softbounce setting. It
also updates translations.  See below for details.

The software is available from:
<http://developer.berlios.de/project/showfiles.php?group_id=1824>

The fetchmail home pages are:
<http://www.fetchmail.info/>  or  <http://fetchmail.berlios.de/>

These are the relevant changes since the previous release.
Unless otherwise noted, changes to this release were made by Matthias Andree:

# REGRESSION FIXES
* The multiline SMTP error fix in release 6.3.12 caused fetchmail to lose
  message codes 400..599 and treat all of these as temporary error. This would
  cause messages to be left on the server even if softbounce was turned off.
  Reported by Thomas Jarosch.
