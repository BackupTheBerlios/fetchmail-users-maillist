> command line, but don't work fine:
>
> % fetchmail -vc --ssl --service 995 --protocol POP3 -u penviarcorreu --auth
> password gmail.com --logfile ./fetchmail.log

You may want to read Google's own documentation
(http://mail.google.com/support/bin/answer.py?answer=13287). The
correct host is not "gmail.com" but "pop.gmail.com". You also don't
need to specify "--service 995".
