> Can I use the option --antispam for this?

That depends on how close to the default configuration your actual
Postfix configuration is. In 2.8.X, antispam 501 should do the job
because "invalid hostname" is the only situation where Postfix should
issue a 501 code.

(Note that for fetchmail versions before 6.3.19, antispam is only
effective in the rcfile, and is ignored on the command line, due to a
bug that got fixed in 6.3.19.)
