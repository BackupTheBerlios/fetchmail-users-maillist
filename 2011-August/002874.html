>  Is there a way to use fetchmail from a machine outside of the department network, thru this "host.dept.fr" e.g.?

     Yes.  Use OpenVPN or SSH.

     If you have root and an available (non-firewalled) port, you can 
set up an OpenVPN server on host.dept.fr and then connect to it from 
outside the network.  This will give you true "remote" access.  You 
should check with your I.T. department first, though, as they probably 
already have a VPN server you can use (or they might have a policy 
against setting up VPNs on your own).

     Since you already have SSH access on host.dept.fr, it would be 
easier to set up an SSH tunnel.  That does not require root.  But, it 
requires that you tell fetchmail to connect to "localhost" when 
accessing it over the SSH tunnel.  Google "ssh tunnel imap" and read any 
of the first four results.

     Since this is networking issue, and not directly related to 
fetchmail or IMAP, please direct any follow-up questions to the 
appropriate OpenVPN or SSH mailing list.
