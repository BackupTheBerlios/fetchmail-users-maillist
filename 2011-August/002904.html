On Sat, Aug 20, 2011 at 20:28, Xavier <fetchmailquestions at gmail.com> wrote:
> Hi to all !
>
> How can I check if fetchmail download the mail of the server

Generally by checking the logs, including your mail server's logs. In
the case of fetchmail it depends on whether you've configured
fetchmail to write to a log file, to syslog or to not log at all. Your
mail server will almost certainly be logging to syslog.

> and where store it ?

That depends on how you configured fetchmail. If you configured it to
hand the email to your local SMTP server then it is decided by your
server's configuration (in particular the configuration of the
IMAP/POP3 server the SMTP server passes the email to for final
delivery). If you've configured it to deliver via an MDA then that
depends on the configuration of the MDA. In neither case does
fetchmail have any control.
