The solution to this particular question is:

1. disable fetchmail's daemon mode (i. e. set daemon interval to 0)
   (the messages get sent only when daemon mode is enabled)

2. run fetchmail as cron job instead
   (to reinstate periodic fetching)

Nonetheless, I'd like to see information per FAQ #G3 so as to figure
what's going on.  Google Apps might be different from Google Mail.
